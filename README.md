# ecommerce-ab-testing
Análisis completo de un experimento A/B para una tienda online, donde se evalúa el impacto de cambios en el proceso de compra sobre las conversiones. Incluye limpieza de datos, validación de grupos, análisis de eventos, verificación de sesgos y realización de pruebas estadísticas para determinar si existe efecto significativo entre grupos A y B.
